## Installation

```bash
# pnpm
pnpm add pinsel

# npm
npm i pinsel
```
*There is also the possibility to use only parts of Pinsel or to write extensions yourself. If you want to know more about this, have a look at the detailed documentation: `@pinsel/core` & `@pinsel/renderer`*
